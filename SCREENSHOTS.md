# 스크린샷 가이드

Play Store 등록을 위한 스크린샷을 Galaxy S24 Ultra에서 촬영해주세요.

## 필수 스크린샷 (최소 2개, 권장 4-8개)

### 요구사항
- 해상도: 1080x1920 ~ 7680x4320
- 형식: PNG 또는 JPG
- 최소 2개, 최대 8개

### 촬영할 화면

1. **메인 화면 (필수)**
   - 전체 게시글 목록이 보이는 화면
   - 여러 사이트의 게시글이 섞여있는 모습

2. **사이트 필터 화면 (필수)**
   - 상단의 사이트 선택 드롭다운을 펼친 화면
   - 또는 특정 사이트를 선택한 화면

3. **다크모드 화면 (권장)**
   - 설정에서 다크모드를 활성화한 화면
   - 게시글 목록이 어두운 테마로 보이는 모습

4. **페이지네이션 화면 (선택)**
   - 하단의 페이지 번호가 보이는 화면

## 촬영 방법

1. Galaxy S24 Ultra에서 앱 실행
2. 각 화면에서 **전원 버튼 + 볼륨 다운** 동시 누름
3. 스크린샷이 `Pictures/Screenshots/` 폴더에 저장됨
4. PC로 파일 전송:
   - USB 케이블 연결
   - 파일 탐색기에서 휴대폰 → Pictures → Screenshots
   - 스크린샷을 PC의 `D:\community-blog\screenshots\` 폴더로 복사

## Play Console 업로드 위치

**프로덕션 → 스토어 등록정보 → 기본 스토어 등록정보 → 그래픽**
- 휴대전화 스크린샷 섹션에 업로드
